# Dog Mouse Pro – Design

- Theme: dark cyberpunk gaming
- Primary color: #00FF41 (neon green matrix)
- Accent: #FF0040 (aggressive red)
- Warning: #FFB800 (amber)
- Background: #0A0A0A (near black)
- Card bg: #111111 / #1A1A1A
- Border: #00FF41 with glow effect
- Font: Orbitron (display/headings), Space Mono (data/stats)
- Border radius: sharp 4px / 8px
- Shadows: neon glow box-shadow in primary/accent colors
- Aesthetic: military HUD + underground hacker tool
