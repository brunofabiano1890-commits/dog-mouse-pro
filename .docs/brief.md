# Dog Mouse Pro
- One-line positioning: App mobile de otimização de mouse/aim para jogos com sistema de ativação por chave e proteção anti-ban
- Target users: Jogadores mobile (Free Fire, PUBG, etc.) que querem vantagem competitiva com sensibilidade de mouse aprimorada
- Core features:
  1. Tela de ativação com chave (activation key system)
  2. Painel de configurações de mouse/aim (sensibilidade, DPI, velocidade)
  3. Sistema anti-ban (randomização de fingerprint, delay humanizado, modo stealth)
  4. Perfil do usuário com status de licença
  5. Toggle de ativação principal (liga/desliga o hack)
  6. Configurações avançadas (gyroscope, recoil control, aim assist)
- Device strategy: mobile_only
- Design style: dark cyberpunk, neon verde/vermelho, gaming agressivo
- Technical constraints: Next.js App Router, localStorage para chaves de ativação
- Nova Agent: not needed
- Completed: imagens geradas
- Current iteration: interface completa v1
